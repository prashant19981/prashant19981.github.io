Spinner

This is a 3D game for which I created the Spinners model using Blender. In order to increase the performance of the game instead of using shaders I used a simple png image as the Spinner top surface. In this game, opponent Spinners are being controlled by the computer which follows 3 states (attack the main player, attack another player, and stay neutral) to make the game feel like a real match. To improve the performance of the game I reduced the vertex count of the Spinner models.
